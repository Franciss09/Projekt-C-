# Projekt-C-
Symulator diety, pomoc w gubieniu kilogramów

Projekt 06. Symulator diet (2 osoby)
Program ma pobierać od użytkownika dane w postaci:
• płeć,
• wiek,
• informacje o trybie życia (siedzący tryb życia, niska aktywność fizyczna, średnia aktywność
fizyczna, wysoka aktywność fizyczna),
• wzrost,
• waga,
• ile kilogramów chce stracić i w ciągu jakiego okresu.
Na tej podstawie powinien być określany miesięczny jadłospis zbilansowany pod względem zapotrzebowania na składniki odżywcze. Program powinien umożliwiać wypisywanie jadłospisu na dowolny okres (dzień, dwa dni, tydzień, miesiąc, itd.). Należy wybrać jedną z dostępnych zbilansowanych diet i podać w projekcie źródło.
